Lab 2: Simple Data Path

-----------
DESCRIPTION

In this lab, the user enters values (0-F) from the keypad into different registers and are displayed. The ALU inputs rotate a certain number
of times based on the user's input and is displayed.

-----------
FILES

-
Lab2.lgi

This file includes the Multimedia Logic file with the circuit.

-----------
INSTRUCTIONS

This program is intended to be run using Multimedia Logic. Enter values into the keypad and chose which values you want to rotate.
