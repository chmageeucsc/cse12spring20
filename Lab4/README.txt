Lab 4: Sorting Integers

-----------
DESCRIPTION

In this lab, the user enters up to 8 program arguments from 0x0 to 0xFFF. The program
    converts these hex to decimal then sorts them from least to greatest.
-----------

This file includes the MARS file with the Sorting Integers program.

This is the pseudocode for the lab in the python file. (NOTE: not actually in python)

-

The block diagram/ flowchart for logic behind program. (Visual psuedocode)

INSTRUCTIONS

This program is intended to be run using MARS. Only enter hex values from 0x0 to 0xFFF.
